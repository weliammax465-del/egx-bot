# 🇪🇬 EGX Technical Analysis Bot — البوت الاحترافي للبورصة المصرية

بوت تليجرام يرسل تقريرًا تقنيًا احترافيًا يوميًا عن أسهم البورصة المصرية (EGX)
باللغة العربية، مع تحليل جميع الأسهم باستخدام 9 مؤشرات تقنية متقدمة.

## المميزات

### 📊 التحليل التقني الاحترافي
- **9 مؤشرات تقنية** لكل سهم:
  - RSI (مؤشر القوة النسبية)
  - Stochastic Oscillator (الاستوكاستك)
  - MACD (ماكد)
  - Bollinger Bands (بولينجر باند)
  - SMA 20/50/200 (المتوسطات المتحركة — Golden Cross / Death Cross)
  - ADX (مؤشر قوة الاتجاه)
  - OBV (مؤشر التوازن الحجمي — تراكم/توزيع)
  - Williams %R (ويليامز)
  - Volume Profile (فوليوم بروفايل — POC، Value Area)
  - ATR (متوسط المدى الحقيقي — التذبذب)
- **نظام نقاط** (Scoring) يجمع إشارات جميع المؤشرات
- **ترتيب الأسهم** من الأكثر صعودًا إلى الأكثر هبوطًا
- **ملخص عربي بالذكاء الاصطناعي** (Gemini 2.5 Flash)

### 🤖 أوامر البوت
| الأمر | الوصف |
|-------|-------|
| `/start` | رسالة ترحيب |
| `/report` | تقرير تحليلي كامل (مسح جميع الأسهم + تحليل AI) |
| `/scan` | مسح سريع لأقوى الأسهم صعودًا وهبوطًا |

### ⏰ الجدولة التلقائية
- **يوميًا الساعة 9:00 صباحًا** بتوقيت القاهرة (قبل فتح السوق)
- أيام التداول: الأحد - الخميس
- يعمل تلقائيًا عبر GitHub Actions — **بدون أي تدخل يدوي**

## الإعداد

### متغيرات البيئة

```bash
export TELEGRAM_BOT_TOKEN="your_bot_token"
export TELEGRAM_CHAT_ID="your_chat_id"
export GEMINI_API_KEY="your_gemini_key"
```

### التشغيل المحلي

```bash
pip install -r requirements.txt
python bot.py              # تشغيل تفاعلي
python bot.py --scheduled  # إرسال تقرير مجدول
```

### النشر على GitHub Actions

1. ارفع الكود إلى GitHub repo
2. أضف Repository Secrets:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `GEMINI_API_KEY`
3. فعّل GitHub Actions
4. التقرير سيُرسل تلقائيًا يوميًا الساعة 9 صباحًا بتوقيت القاهرة
