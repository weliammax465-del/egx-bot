# 🇪🇬 EGX Daily Market Report Bot

بوت تليجرام يرسل تقريرًا يوميًا عن البورصة المصرية (مؤشر EGX 30) باللغة العربية،
مع ملخص ذكي generated by Google Gemini AI.

## المميزات

- **تقرير يومي تلقائي** — مؤشر EGX 30 مع التغيير اليومي والشهري والسنوي
- **ملخص عربي بالذكاء الاصطناعي** — تحليل سوقي موجز بواسطة Gemini 2.5 Flash
- **جدولة تلقائية** — عبر GitHub Actions (مجاني 100%)
- **تنبيهات ذكية** — رسائل عند إغلاق السوق أو تعذّر جلب البيانات
- **إخلاء مسؤولية** — تنبيه واضح أن المعلومات للأغراض المعلوماتية فقط

## المصادر

| المكون | المصدر | التكلفة |
|--------|--------|---------|
| بيانات السوق | [Trading Economics](https://tradingeconomics.com/egypt/stock-market) | مجاني |
| الملخص الذكي | Google Gemini 2.5 Flash (free tier) | مجاني |
| التوصيل | Telegram Bot API | مجاني |
| الجدولة | GitHub Actions | مجاني |

## الإعداد

### 1. متطلبات

- Python 3.11+
- Telegram Bot Token (من [@BotFather](https://t.me/BotFather))
- Gemini API Key (من [Google AI Studio](https://aistudio.google.com/app/apikey))
- Chat ID بتاعك (ابعت رسالة للبوت واستخدم getUpdates)

### 2. التثبيت المحلي

```bash
git clone https://github.com/weliammax465-del/egx-bot.git
cd egx-bot
pip install -r requirements.txt
```

### 3. متغيرات البيئة

```bash
export TELEGRAM_BOT_TOKEN="your_bot_token"
export TELEGRAM_CHAT_ID="your_chat_id"
export GEMINI_API_KEY="your_gemini_key"
```

### 4. التشغيل

```bash
# تشغيل البوت تفاعليًا (مع polling)
python bot.py

# إرسال تقرير مجدول (للاستخدام مع GitHub Actions)
python bot.py --scheduled
```

### 5. النشر على GitHub Actions

1. ارفع الكود إلى GitHub repo
2. أضف Repository Secrets:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `GEMINI_API_KEY`
3. فعّل GitHub Actions من تبويب Actions
4. التقرير هيتوصّل تلقائيًا أيام التداول (الأحد - الخميس) الساعة 10:30 صباحًا بتوقيت القاهرة

## أيام تداول البورصة المصرية

البورصة المصرية تعمل من **الأحد إلى الخميس** (الجمعة والسبت عطلة).
البوت يتخطى أيام العطلة تلقائيًا لتوفير استهلاك الـ API.
